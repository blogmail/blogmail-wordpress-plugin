This plugin integrates blogmail into your WordPress posts.

== Description ==

[blogmail](https://blogmail.co/) provides privacy-focused newsletter or
mailing list support for bloggers. blogmail uses your blog's RSS feed to
determine when new posts are published, and your subscribers are notified via
email.

== Installation ==

To get started, [sign up here](https://blogmail.co/signup) and create a
newsletter. After the plugin is installed, go to Settings -> blogmail and
enter the newsletter ID into the plugin settings and save.

Add the RSS source for your blog to the [blogmail
dashboard](https://blogmail.co/dashboard). By default this
will be `https://<your-site-url>/?feed=rss`.
